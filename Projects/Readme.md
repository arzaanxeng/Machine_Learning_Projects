# Machine Learning Projects: Basics to Advanced

A collection of Machine Learning projects built while progressing from fundamental concepts to advanced AI systems. This repository focuses on understanding the complete ML pipeline, including data preprocessing, model development, evaluation, optimization, and deployment.

## Project Roadmap

### Beginner

* Linear Regression from Scratch
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Trees
* Data Analysis & Visualization with Pandas and Matplotlib

### Intermediate

* Random Forests
* Support Vector Machines (SVM)
* Feature Engineering & Selection
* Hyperparameter Tuning
* Customer Segmentation using Clustering

### Advanced

* Neural Networks from Scratch (NumPy)
* Deep Learning with PyTorch
* Convolutional Neural Networks (CNNs)
* Natural Language Processing (NLP)
* Recommendation Systems
* End-to-End ML Pipelines

### AI & LLM Engineering

* Transformer Architecture Fundamentals
* Fine-Tuning Pretrained Models
* Retrieval-Augmented Generation (RAG)
* LLM Applications
* AI Agent Development

## Tech Stack

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* PyTorch

## Goal

To document the journey from learning core Machine Learning concepts to building production-ready AI systems and Large Language Model applications.
